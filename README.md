# COMANDOS LINUX

## virar sudo:
```sh
sudo -i
```

## verificar ip:
```sh
ip a
```

## atualizar pacotes do sistema operacional:
```sh
sudo apt-get update && sudo apt-get upgrade -y
```

## reiniciar o sistema:
```sh
reboot
```

## limpar a tela:
```sh
ctrl l / clear
```

## sair do fim:
```sh
ctrl c
```

## mostrar o diretorio atual:
```sh
pwd
```
